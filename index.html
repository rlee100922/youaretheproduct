    MARKETPLACE OF _™  ·  CRITICAL STATEMENT

    About This Marketplace

    What Did You Just Experience?

    A Project Statement

    Marketplace of __™

    A campaign website that visualizes how social media platforms transform human attention, behavior, and interaction into economic value. The blank in the title represents you — the unnamed product at the center of the system.

    B Why E-Commerce Language?

    The Language of Valuation

    • Platforms already treat users as assets — this project makes that logic visible.
    • E-commerce framing (SKU, daily price, tier, resale value) mirrors how behavioral data is actually packaged and sold.
    • Experiencing yourself as a product listing is harder to dismiss than reading about it.
    C Concepts Behind the Project

    Theoretical Framework

    Attention Economy

    Human attention is a scarce, tradeable resource.

    Surveillance Capitalism

    Behavioral data is captured and sold as a commodity.

    Digital Panopticon

    Algorithmic systems create constant, invisible visibility.

    FOMO

    Engineered anxiety that keeps users returning and producing.

    Enshittification

    Platforms degrade user experience to maximize advertiser extraction.

    Self-Exploitation

    Users perform, post, and engage for systems that profit from it.

    This project is not saying "delete social media." It tries to make the hidden system visible — how every scroll, like, and post becomes measurable data and market value. The user is not the customer. The user is the product.
